Movie_API


Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Supported Requests

Request URL HTTP Method Request Body Data Format Response Body Data Format
Get List of Movies /movies GET None An array of JSON objects with data about all movies
Get List of Movies by Director /movies/directors/:directorName GET None An array of JSON objects with data from all movies by requested director
Get List of Movies by Genre /movies/genres/:genreName GET None An array of JSON objects with data about all the movies by requested genre
Get Information about a Movie by Title /movies/:Title GET None A JSON object with data about the requested movie like this:

'Birthyear': 'Year'},
'ImagePath': 'image.png',
'Featured': 'true' }

Remove a User by Username /users/:Username DELETE None A message confirming that a user was deleted.
